

***************************** Carrom Board Game **************************************************

1) Board of 6*6 square
2) striker control --> space bar 
3) powerbar or speed meter control --> arrow Up & arrow Down
4) striker reset after one strike control --> 'b'
5) pointer directions controls ( 'w' -> up , 'a' -> left , 's' -> down , 'd' -> right)
6) mouse pointer at any position by clicking button left down and striker moves after button left up
7) mouse power pointer -> button right up and down 

compiling the file of cpp  "g++ -std=c++11 *.cpp -lglut -lGL -lGLU -g"
Output -> ./a.out or make then ./carrom
